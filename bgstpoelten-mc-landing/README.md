# Minecraft Landing Page - BGST Pölten

This is a landing page for the official Minecraft server of BGST Pölten school.

## Features

- Modern, Minecraft-themed design with green color scheme
- Server information display
- Copy-to-clipboard functionality for the server address
- Connection instructions
- Server rules
- Responsive design for mobile devices

## Files

- `index.html` - Main HTML structure
- `styles.css` - CSS styling with Minecraft-inspired theme
- `script.js` - JavaScript functionality for copying server address

## How to Use

To serve this page on your web server, simply copy all files to your web directory. 
The page has been designed to work with the BGST Pölten Minecraft server at bgstpoelten.ac.at.

## Customization

To customize for your server, update the following in the HTML:
- Server address
- Server version
- Game mode
- Rules
- Contact information
