# Minecraft SaaS Platform

Welcome to the Minecraft SaaS Platform.

## Overview

This is a comprehensive platform for managing multiple Minecraft servers with a centralized Admin API and a Vue.js frontend.

### Key Features
*   **Universal Base Image**: Single Docker image for all PaperMC versions.
*   **Centralized Management**: Manage multiple servers via a single interface.
*   **Real-time Statistics**: In-memory history service with public API access.
*   **Interactive Dashboard**: 3D performance charts using Three.js.
*   **Public API**: Read-only endpoints for server status and datapack info.

For detailed information about the project, including setup, architecture, and development processes, please refer to the documentation in the `docs` directory.
