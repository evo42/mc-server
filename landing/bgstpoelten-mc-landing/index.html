Minecraft Server

BGST Pölten

Willkommen auf dem offiziellen Minecraft Server der BGST Pölten!

Verbinde dich mit deinen Mitschülern und Lehrern in einer riesigen Minecraft-Welt.

Server Information

Server Adresse: bgstpoelten.ac.at

Version: 1.21.1 (oder neuer)

Spielmodus: Survival mit Schulregeln

Wie verbindest du dich?

  1. Öffne Minecraft auf deinem Computer
  2. Klicke auf "Mehrspieler"
  3. Klicke auf "Server hinzufügen"
  4. Gebe die Serveradresse bgstpoelten.ac.at ein
  5. Klicke auf "Fertig"
  6. Klicke auf den Server, um zu verbinden

Spielregeln

  • Sei respektvoll zu anderen Spielern
  • Kein Griefing ohne Erlaubnis
  • Kein Cheatbenutzung
  • Baue nur an erlaubten Orten
  • Melde Probleme an die Server-Admins
